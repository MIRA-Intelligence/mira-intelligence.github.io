为科学研究设计的 AI Agent

· 让 Agent 服务于科研

MIRA 是一个开源的项目级 AI 助手。把"研究 → 实验 → 报告"这条链路交给 Agent，输入一个目标，输出可复现的实验、可读的报告、可分享的 PPT。
配套专属桌面 UI、CLI、PyPI包、Docker镜像，飞书 / Slack / QQ 一并接入。

核心能力

为复杂、长程的研究任务而生

不是又一个聊天机器人。MIRA 把"想清楚 → 跑实验 → 写报告"端到端做完，每一步都留下证据。

🧪

项目级 Research Agent

围绕 workspace / project / experiment / task_plan 五要素建模，让 Agent 长期协作而不是一次性聊天。每一步实验都有 task_plan.json 记录，可复现、可审计。

→
🤖

20+ LLM Provider，开箱即用

OpenAI、Anthropic、OpenRouter、DeepSeek、Azure、Ollama、vLLM 等全部内置。支持 routeByComplexity，便宜的活给 small 模型，难的给 large。

→
🖥️

CLI / Web / Desktop 三形态

命令行跑实验，浏览器看进度，Electron 桌面端开机即用。三者共用一份配置，状态实时同步。

→
💬

13 个 Channel 接入

飞书、Slack、Telegram、钉钉、邮件、Matrix…… 在群里 @ 一下让 Agent 跑实验、出报告、回复结论，不用打开浏览器。

→
🛡️

Guardrail 自动修复

实验字段缺失？模型自己回头补。最多 3 次自动修复回合，失败再交给人工。strict 模式可强制接合规字段。

→
📦

本地优先，自托管友好

所有数据落在 ~/.mira/，单机 / 团队 docker-compose / 远程 gateway 三种部署模式都覆盖。零厂商锁定。

→
桌面 UI

一个为科研 Agent 而生的工作台

MiraUI 把项目队列、阶段流水线、实验详情、Result 导出中心整合在一个 Electron 应用里。Agent 在后台跑，你随时切换项目看进度，点几下就能导出 PPT、论文、experiment_report。

  • 实时同步：WebSocket 推 + REST 兜底，实验状态实时更新。
  • Web/Desktop 一份代码：浏览器开发，Electron 打包，打通前端后端。
  • 可暂停 / 可回滚：每个实验的 prompt、tool 调用、结果都有完整 trace。
